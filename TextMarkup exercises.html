Solving Equations in Two Variables with Python

An equation is said to be linear equation in two variables if it is written in the form of ax + by + c=0, where a, b & c are real numbers and the coefficients of x and y,
i.e a and b respectively, are not equal to zero.
For example, 10x+4y = 3 and -x+5y = 2 are linear equations in two variables. The solution for such an equation is a pair of values, one for x and one for y which
further makes the two sides of an equation equal.

The solution of linear equations in two variables, ax+by = c, is a particular point in the graph, such that when x-coordinate is multiplied by a and y-coordinate
is multiplied by b, then the sum of these two values will be equal to c. Basically, for linear equation in two variables,
there are infinitely many solutions.

Understanding how to solve equations in two variables is an important skill in algebra. Let's go through the process step by step and then implement a simple Python
program to solve such equations.

Step-by-Step Guide:

  1. Identify the Equations: Start with a system of two equations in the form (ax + by = c).
  2. Select a Method: There are various methods to solve such systems. Here, we'll focus on substitution.
  3. Solve for One Variable: Choose one equation and solve it for one variable in terms of the other.
  4. Substitute: Substitute the expression obtained in the previous step into the other equation.
  5. Solve: Solve the resulting equation to find the value of the variable.
  6. Back-Substitute: Substitute the found value back into one of the original equations to find the other variable.
  7. Verify: Verify the solution by checking it in both original equations.

Example:

Consider the following system of equations:

(2x + y = 5)
(3x - 2y = 8)

Choose the first equation and solve for y:

(2x + y = 5)
(y = 5 - 2x)

Substitute (y = 5 - 2x) into the second equation:

((3x - 2(5 - 2x)) = 8)

Solving for x gives (x = 2). Substitute (x = 2) back into the expression for y: (x = 1).

(2(2) + y = 5)

Solving for y, we get (y = 1). So, the solution to the system is (x = 2) and (y = 1).

Python Program:

Now, let's implement a Python program that solves equations in two variables:

        
            # Coefficients
            a1, b1, c1 = 2, 1, 5
            a2, b2, c2 = 3, -2, 8

            # Solve for y in terms of x
            y_in_terms_of_x = c1 - a1 * x

            # Substitute y_in_terms_of_x into the second equation
            equation = a2 * x + b2 * y_in_terms_of_x - c2

            # Solve the resulting equation for x
            solution_x = equation / (a2 + b2)

            # Substitute x back into y_in_terms_of_x to find y
            solution_y = y_in_terms_of_x.subs(x, solution_x)

            # Verify the solution by checking it in both equations
            check1 = a1 * solution_x + b1 * solution_y == c1
            check2 = a2 * solution_x + b2 * solution_y == c2

            print("Solution:", solution_x, solution_y)
            print("Verification:", check1 and check2)

This Python program is a basic example. In real-world scenarios, consider using specialized libraries or more robust methods for solving equations.
