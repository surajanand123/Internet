2. Use definition list to create a list that defines p, sub, sup, code, pre, img, a tags.

HTML Tag Definitions

<p>
The paragraph tag is used to define and structure text as paragraphs.
<sub>
The subscript tag is used to represent subscript text, typically rendered lower and smaller than the surrounding text.
<sup>
The superscript tag is used to represent superscript text, typically rendered higher and smaller than the surrounding text.
<code>
The code tag is used to define a piece of computer code.
<pre>
The preformatted tag is used to preserve both spaces and line breaks within the enclosed text.
<img>
The image tag is used to embed images in an HTML document.
<a>
The anchor tag is used to create hyperlinks, allowing navigation to other web pages or resources.
